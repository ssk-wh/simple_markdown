# SVG 图片显示支持

## 背景
当前 ImageCache 使用 `QPixmap::load()` / `QPixmap::loadFromData()` 加载图片，不支持 SVG 格式。Markdown 文档中的 `![](diagram.svg)` 无法显示。

## 目标
- [ ] 本地 .svg 文件能正常渲染显示
- [ ] 网络 SVG URL 能异步下载并渲染
- [ ] data URI 中的 `image/svg+xml` 能解码显示
- [ ] SVG 尺寸：优先使用 SVG 内部 viewBox/viewport 尺寸，无 intrinsic size 时使用默认 800×600
- [ ] SVG 在 getSize 中能返回正确尺寸供 layout 阶段计算高度
- [ ] 单元测试覆盖 SVG 加载场景

## 设计
在 ImageCache 中新增 SVG 渲染路径：
- 检测方式：文件扩展名 `.svg`/`.svgz`、Content-Type `image/svg+xml`
- 渲染方式：`QSvgRenderer` → `QImage`(ARGB32) → `QPixmap`
- 尺寸获取：`QSvgRenderer::defaultSize()`，无 intrinsic size 时默认 800×600
- 构建依赖：新增 `Qt5::Svg` 模块
