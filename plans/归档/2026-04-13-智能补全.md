# Markdown 智能补全

## 背景
编辑长文档时缺少语法和路径补全，用户需手动输入常见结构（链接、图片、表格骨架等）。

## 动作
- [ ] 触发字符设计：`[`、`!`、`|`、`#`、`/` 等
- [ ] Markdown 语法片段补全（链接 / 图片 / 表格 / 代码块）
- [ ] 链接路径补全：扫描当前文档目录下的 .md 文件
- [ ] 图片路径补全：扫描相对路径下的图片资源
- [ ] 标题锚点补全（`#锚点`）：从当前文档已解析的标题列表生成
- [ ] 配置开关（默认开），ESC 关闭当前补全

## 验收
- 输入 `[` 后能浮现链接片段
- 输入 `](./` 后能列出当前目录 .md 文件
- 输入 `![](./` 后能列出图片
- 关闭开关后无任何弹窗
