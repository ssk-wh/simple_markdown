# 仅预览模式下隐藏左下角行号列号

## 背景

用户反馈（2026-05-13）：

> 如果只有预览区在显示，左下角的行多少和列多少就没必要显示了，这部分说的是编辑区，此时显示没必要

应用主窗口底部（或某处）显示编辑器侧的「行 X / 列 Y」状态——这是**编辑光标位置**的实时反馈。
当「视图 → 显示区域」切到「仅预览」（1.0.1 引入）时编辑区不可见，行/列信息失去
参考意义。

## 动作

- [ ] 找到行/列显示的 widget（可能在 MainWindow status bar 或自定义 widget）
- [ ] 监听「显示区域」变化信号（仅编辑器 / 仅预览 / 编辑器+预览 三种模式）
- [ ] 在「仅预览」模式时 setVisible(false)，其他两种模式 setVisible(true)
- [ ] 演示模式（F11，仅预览）也应同步隐藏——产品愿景 §4 已说 F11 是仅预览

## 验收

- 视图 → 显示区域 → 仅预览 → 左下角行/列消失
- 切回 → 仅编辑器 或 编辑器+预览 → 重新显示
- F11 演示模式同步隐藏，退出 F11 恢复
- 状态切换无闪烁、无残留布局占位

## 优先级

P2（小型 UX 打磨，单 widget visibility 控制）。
