# TOC 目录节点支持折叠

## 背景

Liquid Glass mockup（`tmp/theme_mocks/mock_*.html`）里的 TOC 面板展示了折叠箭头（▸ / ▾）。
当前 `TocPanel` 直接扁平列出所有标题，深层文档（h1..h4 多层嵌套）下 TOC 会很长、不好导航。

## 动作

- [ ] `TocPanel` 支持按层级嵌套的 tree model
- [ ] 每个父节点加"折叠箭头"图标（可点击切换展开/收起）
- [ ] 折叠状态持久化（同一文件再次打开时保持）
- [ ] 键盘快捷键：← 收起当前节点，→ 展开当前节点
- [ ] Spec 更新：`specs/模块-preview/08-TOC面板.md` 新增 INV + T 条目

## 验收

- h1-h4 多层嵌套文档 TOC 可按层级折叠
- 折叠状态随文件持久化，重开文件恢复
- 折叠/展开操作不影响预览区滚动位置
