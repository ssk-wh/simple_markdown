# 启动窗口几何

## 1. 范围

本 Spec 定义 SimpleMarkdown 主窗口启动时的窗口尺寸与状态（最大化/普通），以及会话间的几何持久化。

不涉及：演示模式（见 11-演示模式.md，使用 `showFullScreen()`）、专注模式、子对话框几何。

## 2. 不变量

### INV-LAUNCH-MAXIMIZED-DEFAULT
首次启动（QSettings 中无 `window/geometry` 键），主窗口必须以**最大化**状态显示。

理由：现代桌面应用大多 23"+ 屏幕，1280×800 默认尺寸在大屏上显得局促；用户期望首次启动即可使用全屏空间，而非手工最大化一次。

### INV-LAUNCH-RESPECT-SAVED
存在 `window/geometry` 键时，启动时必须用 `QWidget::restoreGeometry()` 恢复用户上次关闭时的几何，**包括最大化标志**。即：

- 用户上次关闭时是最大化 → 本次启动最大化
- 用户上次关闭时是普通窗口（手工 resize 过） → 本次启动恢复同位置同尺寸

理由：用户偏好优先；若用户上次主动取消最大化并调整到特定大小，本次必须尊重这个选择，否则等价于"忽略用户行为强制覆盖"。

### INV-LAUNCH-NORMAL-FALLBACK
即使默认最大化，构造函数中仍必须调用 `resize(默认宽, 默认高)`（当前为 1280×800）。该尺寸作为：

1. 用户从最大化状态切到普通状态时的初始还原尺寸（首次启动场景）
2. `restoreGeometry` 失败时的兜底

不得删除该行。

### INV-LAUNCH-NO-DOUBLE-SHOW
禁止在 `main.cpp` 中**同时**调用 `show()` 和 `showMaximized()`。两者会触发两次窗口显示事件，导致 `showEvent` 内的 splitter 恢复逻辑被执行两次。必须用 `setWindowState(Qt::WindowMaximized)` + `show()`，或单独调用 `showMaximized()` 之一。

## 3. 当前实现

| 关注点 | 位置 | 说明 |
|--------|------|------|
| 默认尺寸 | `MainWindow::MainWindow` ctor `resize(1280, 800)` | 普通状态还原尺寸 |
| 默认最大化 | `MainWindow::MainWindow` ctor `setWindowState(Qt::WindowMaximized)` | 标记，等到 show() 时生效 |
| 恢复几何 | `MainWindow::loadSettings` `restoreGeometry(s.value("window/geometry"))` | 若存在则覆盖 ctor 的默认 |
| 保存几何 | `MainWindow::saveSettings` `s.setValue("window/geometry", saveGeometry())` | 在 closeEvent 中调用 |
| 显示 | `main.cpp` `window.show()` | 单次调用 |

## 4. 时序

```
ctor:
  resize(1280, 800)            # INV-LAUNCH-NORMAL-FALLBACK
  setWindowState(WindowMaximized)  # INV-LAUNCH-MAXIMIZED-DEFAULT
  ...
  loadSettings():
    if QSettings has window/geometry:
      restoreGeometry(...)     # INV-LAUNCH-RESPECT-SAVED 覆盖默认
    # 否则保留 ctor 中的最大化标志

main.cpp:
  window.show()                # INV-LAUNCH-NO-DOUBLE-SHOW
```

## 5. 验收条件（T 编号）

| 编号 | 场景 | 验证 |
|------|------|------|
| T1 | 清空 settings 首次启动 | 窗口呈最大化状态（无任务栏遮挡，覆盖屏幕除任务栏外区域） |
| T2 | 上次最大化关闭 → 重启 | 仍最大化 |
| T3 | 上次手工取消最大化并 resize 为某尺寸 → 关闭 → 重启 | 恢复同位置同尺寸的普通窗口（不强制最大化） |
| T4 | 启动后从最大化双击标题栏切回普通 | 还原到 1280×800（首次启动场景）或上次保存的普通尺寸 |
| T5 | 不在 main.cpp 中调用两次 show*()  | 代码审查 + 启动时 showEvent 仅触发一次 |

## 6. 反例（已知陷阱）

- ❌ 在 main.cpp 用 `window.showMaximized()` 替换 `window.show()`：若 settings 中保存了"普通窗口"几何，showMaximized 会强制最大化，违反 INV-LAUNCH-RESPECT-SAVED
- ❌ 同时调用 `restoreGeometry` 和 `showMaximized`：覆盖关系不明确；当前实现依赖 ctor 中先 set 默认 + loadSettings 中 restoreGeometry 的覆盖语义
- ❌ 删除 ctor 的 `resize(1280, 800)`：违反 INV-LAUNCH-NORMAL-FALLBACK
