# SimpleMarkdown 需求文档

## 1. 产品概述

SimpleMarkdown 是一款轻量、高性能的跨平台 Markdown 阅读/编辑器，支持 Windows 和 Linux。

## 2. 核心功能

### 2.1 编辑模式
- 分屏模式：左侧 Markdown 源码编辑，右侧实时渲染预览

### 2.2 编辑器功能
- 基于 QPainter 的全自绘文本编辑器
- 行号显示
- Markdown 语法高亮（标题、粗体、斜体、代码、链接、列表、引用）
- 光标闪烁、选区高亮
- 键盘快捷键：Ctrl+Z/Y（撤销/重做）、Ctrl+X/C/V（剪切/复制/粘贴）、Ctrl+A（全选）
- Tab 缩进/Shift+Tab 反缩进
- Enter 自动缩进
- 搜索替换（Ctrl+F/Ctrl+H）
- IME 中文输入支持

### 2.3 预览功能
- 基于 QPainter 的全自绘 Markdown 预览渲染
- 支持 Markdown 元素：标题（H1-H6）、段落、粗体、斜体、行内代码、围栏代码块、引用块、有序/无序列表、表格（GFM）、图片（本地）、分割线、链接、删除线
- 编辑后 30ms 防抖自动更新预览

### 2.4 多 Tab 支持
- 同时打开多个文件
- Tab 标题显示文件名和修改标记
- 关闭修改过的 Tab 时提示保存

### 2.5 文件操作
- 新建、打开、保存、另存为
- 拖拽 .md/.markdown/.txt 文件到窗口自动打开
- 最近文件列表（最多 10 个）

### 2.6 主题
- 亮色主题
- 暗色主题
- 菜单切换

### 2.7 编辑器-预览同步
- 编辑器滚动时预览自动跟随

## 3. 性能指标

| 指标 | 目标 |
|------|------|
| 冷启动 | < 0.5 秒 |
| 空载内存 | < 30 MB |
| 打字延迟 | < 3 ms |
| 预览更新 | < 80 ms（含 30ms 防抖）|

## 4. 跨平台支持

- Windows 10/11（MSVC 2019+）
- Linux（GCC 9+ / Clang 10+）

## 5. 技术约束

- C++17
- Qt 5.15
- CMake >= 3.16
