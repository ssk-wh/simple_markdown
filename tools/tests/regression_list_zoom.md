# 列表缩放回归样本

> 防御：Ctrl + 鼠标滚轮缩放预览字号时，列表项序号位置/字号未跟随重算，
> 出现序号被内容遮挡或与第一行错位的现象。
> 验收要点：在不同缩放级别下分别 dump `render_blocks.json`，「序号遮挡」「字体一致性」
> 「字体层级」三项检查均需通过。

## 长项内容 + 加粗前缀（容易出现序号被覆盖）

- **AI 智能识别**：手写中英文识别、数学/化学公式识别、图形识别、AI 自由书写（表格/流程图）
- **学科工具**：150+ 种教学工具（几何尺规、化学器材、函数图像、计数器、钟表等）
- **课件互动**：投票、随机点名、抢答、计时、白板批注、屏幕广播、远程控制
