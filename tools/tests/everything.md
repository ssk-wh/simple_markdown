# 代码块

## 围栏代码块（C++）

```cpp
#include <iostream>
#include <vector>

int main() {
    std::vector<int> nums = {1, 2, 3, 4, 5};
    for (auto n : nums) {
        std::cout << n << " ";
    }
    return 0;
}
```

## 围栏代码块（Python）

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
