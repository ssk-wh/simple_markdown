# 嵌套列表回归样本

> 防御：嵌套有序/无序列表中子项坐标"双倍偏移"导致序号被遮挡或缩进错位。
> 验收要点：list 子项 `x` 坐标应当一致（参见 `verify_render.py` 的「对齐验证」），且
> 序号 `bullet_x + bullet_width <= 子项 x`（参见「序号遮挡」检查）。

## 性能优化的三大支柱

1. **PrintService 优化**（-5-10% 缩略图时间）
   - IPC 通信效率提升
   - 消息队列处理优化
   - 渲染管道加速
2. **缓存策略**（-30% 内存占用）
   - 限制缓存大小（LRU 淘汰）
   - 懒加载 vs 预加载平衡
   - 及时释放不用的数据
3. **异步化**（-0% CPU 峰值）
   - 后台线程渲染
   - 异步 IPC 通信
   - 非阻塞清理

## 模块职责（短列表 + 加粗）

- **DrawBoard**：基于 Qt Graphics Framework 的渲染和交互层
- **VirtualCanvas**：平台无关的画布数据模型和序列化层
